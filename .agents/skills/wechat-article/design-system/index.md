Design System 设计系统规范

基于 12 个顶级科技公司/产品设计灵感的设计参考。

## 主题列表

| Theme | 灵感来源 | 风格定位 |
|-------|----------|----------|
| `tech` | SpaceX | 深空黑 + 大写字母-spacing 工业风 |
| `minimal` | Vercel | 白底极简 + 负字间距 + shadow-border |
| `business` | Stripe | 深海军蓝 + 蓝调阴影 + 轻字重 |
| `claude` | Anthropic Claude | 羊皮纸暖色 + 衬线标题 + 陶土色 |
| `standard` | 通用标准 | 简洁商务 |
| `neo-brutalist` | 原始粗野主义 | 高对比 + 强边框 |
| `dark-finance` | 深色金融 | 深色背景 + 霓虹强调 |
| `luxury-editorial` | 奢侈品牌 | 优雅排版 + 精致留白 |
| `japanese-zen` | 日式侘寂 | 极简禅意 + 柔和色调 |
| `vintage-newspaper` | 复古报纸 | 经典印刷 + 衬线字体 |
| `notion` | Notion | 暖白中性 + 负字间距 + 极细边框 |

## 核心规范

1. **颜色** - 每个主题定义完整的调色板 (Primary/Interactive/Neutral/Surface)
2. **排版** - 字体族 + 字重 + 行高 + 字间距层级表
3. **组件** - 按钮/卡片/输入框/导航样式规范
4. **布局** - 间距系统 + 网格 + 留白哲学
5. **深度** - 阴影层级与 elevation
6. **响应式** - 断点 + 折叠策略
