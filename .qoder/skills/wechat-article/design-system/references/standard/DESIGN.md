# Design System Inspired by predict.html

## 1. Visual Theme & Atmosphere

这是一篇面向微信公众号的财经类文章设计系统。整体风格融合了金融专业感与现代阅读体验，采用**绿色+金色**的双主色调组合，绿色传递专业可信的金融气质，金色点缀则增添活力与重点突出。

页面采用**卡片式布局**，内容区域最大宽度720px，居中展示于浅灰背景上，形成清晰的视觉层次。文章结构通过**渐变头部横幅**、**左侧色块标注框**、**色块装饰标题**等组件建立层次分明的阅读节奏。

设计的一大特色是**信息密度与呼吸感的平衡**：正文行高1.75、字号16px保证舒适阅读；章节间距40px留白充足；表格采用斑马纹与圆角设计，既呈现数据又不显枯燥。

**Key Characteristics:**
- 双主色系统：绿色(#05c15f/#049a4d) + 金色(#fdbb2d)
- 卡片式内容容器，白色背景配淡阴影
- 渐变头部横幅（135deg 线性渐变）
- 左侧色块强调框（核心概念/重要公式）
- 章节标题带竖色块装饰
- 标签式关键词高亮（圆角背景标签）
- 渐变表头 + 斑马纹表格
- 风险提示专用样式（浅红背景）

## 2. Color Palette & Roles

### Primary Colors
- **主绿色** (`#05c15f`): 标题文字、链接、强调元素
- **深绿色** (`#049a4d`): 标题装饰色块、渐变起始色
- **金黄色** (`#fdbb2d`): 渐变终止色、强调框边框、重点点缀

### Backgrounds
- **页面背景** (`#f5f5f5`): 浅灰色，营造卡片浮起感
- **卡片背景** (`#ffffff`): 纯白，内容区域
- **暖白背景** (`#fdfbf7`): 概念框、公式框背景，带轻微暖调
- **浅蓝灰背景** (`#f0f4f8`): 公式框背景
- **浅红背景** (`#fff5f5`): 风险提示框专用

### Text Colors
- **正文** (`#333`): 主要文字
- **次要文字** (`#444`): 概念框文字
- **辅助文字** (`#555`): 表格内容
- **占位文字** (`#777`): 图片占位符
- **说明文字** (`#999`): 图片标题、页脚

### Surface & Effects
- **阴影** (`box-shadow: 0 0 20px rgba(0,0,0,0.05)`): 卡片容器柔和阴影
- **表格阴影** (`box-shadow: 0 2px 8px rgba(0,0,0,0.05)`): 表格微阴影

### Gradient System
- **头部渐变**: `linear-gradient(135deg, #05c15f, #049a4d, #fdbb2d)`
  - 起始：主绿色 → 深绿色 → 金黄色
  - 角度：135度（左上到右下对角线）
  - 应用于头部横幅背景
- **表头渐变**: `linear-gradient(135deg, #05c15f, #049a4d)`
  - 起始：主绿色 → 深绿色
  - 应用于表格头部背景

## 3. Typography Rules

### Font Family
- **Primary**: `-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif`
- **Monospace**: `'Courier New', monospace` (公式专用)

### Hierarchy

| Role | Size | Weight | Line Height | Notes |
|------|------|--------|-------------|-------|
| 主标题 (h1) | 26px | 800 | - | 头部横幅内，带文字阴影 |
| 副标题 | 14px | 300 | - | 头部横幅内，opacity: 0.9 |
| 章节标题 (h2) | 20px | - | - | 主绿色，带下划线装饰 |
| 小节标题 (h3) | 17px | 700 | - | 深色，粗体 |
| 正文 | 16px | - | 1.75 | 基础字号，舒适行高 |
| 概念框正文 | 15px | - | - | 略小于正文 |
| 图片标题 | 12px | - | - | 居中，灰色 |
| 页脚文字 | 12px | - | - | 居中，灰色 |
| 标签文字 | 12px | 600 | - | 圆角标签内 |

### Principles
- **舒适行高**: 1.75的行高为中文长文阅读提供充足呼吸空间
- **字重对比**: 主标题800（ExtraBold），小节标题700（Bold），形成明确层次
- **副标题轻量**: 副标题使用300（Light）字重，与主标题形成视觉对比
- **字号递进**: 26px → 20px → 17px → 16px，层级清晰
- **抗锯齿**: 启用 `-webkit-font-smoothing: antialiased` 优化字体渲染

## 4. Component Stylings

### 头部横幅 (Header Banner)
- 背景：135度渐变（#05c15f → #049a4d → #fdbb2d）
- 文字：白色，居中对齐
- 内边距：40px 上下，30px 下
- 圆角：8px
- 下边距：30px
- 文字阴影：`0 2px 4px rgba(0,0,0,0.2)` 增强可读性
- 特点：渐变从绿色过渡到金色，形成专业而活泼的视觉效果

### 核心概念框 (Concept Box)
- 背景：#fdfbf7（暖白色）
- 左边框：5px solid #fdbb2d（金色强调）
- 内边距：15px 20px
- 圆角：4px
- 字号：15px
- 下边距：30px
- 特点：左侧色块形成视觉锚点，暖色背景区隔正文

### 公式框 (Formula Box)
- 背景：#f0f4f8（浅蓝灰）
- 左边框：5px solid #05c15f（绿色强调）
- 内边距：15px 20px
- 圆角：4px
- 公式样式：居中，等宽字体，上下间距10px
- 特点：与概念框结构相同，但配色区分不同内容类型

### 章节标题 (Section Heading - h2)
- 颜色：#05c15f（主绿色）
- 下边框：2px solid #eee
- 下内边距：10px
- 上下边距：40px 0 20px
- 布局：`display: flex; align-items: center`
- 装饰色块：
  - 宽度：8px
  - 高度：24px
  - 背景：#049a4d（深绿色）
  - 右边距：12px
  - 圆角：2px
- 特点：竖向色块作为章节视觉标记，比单纯文字更醒目

### 标签样式 (Tags)
- 显示：`display: inline-block`
- 内边距：2px 8px
- 背景：#eef2f7（浅灰蓝）
- 文字颜色：#05c15f（主绿色）
- 圆角：4px
- 字号：12px
- 右边距：5px
- 字重：600
- 特点：为关键词提供醒目的视觉标识

### 表格样式 (Table)
- 宽度：100%
- 边框合并：collapse
- 上下边距：25px
- 字号：14px
- 阴影：`0 2px 8px rgba(0,0,0,0.05)`
- 圆角：8px（需配合 overflow: hidden）
- 表头：
  - 背景：渐变（#05c15f → #049a4d）
  - 文字：白色，600字重
  - 内边距：12px 10px
- 表格行：
  - 奇数行：#f9f9f9
  - 偶数行：#ffffff
  - 下边框：1px solid #eee
  - 文字颜色：#555

### 图片容器 (Image Container)
- 宽度：100%
- 高度：200px
- 背景：#e0e0e0（占位色）
- 圆角：8px
- 上下边距：20px
- 图片样式：`width: 100%; height: 100%; object-fit: cover`
- 图片标题：
  - 居中对齐
  - 字号：12px
  - 颜色：#999
  - 上边距：-15px（上移贴近图片）
  - 下边距：20px

### 风险提示框 (Risk Warning)
- 背景：#fff5f5（浅红）
- 边框：1px solid #ffcdd2
- 文字颜色：#c62828（红色）
- 内边距：15px
- 圆角：8px
- 字号：13px
- 上边距：40px
- 特点：使用警告色系，配合图标，视觉上醒目但不突兀

### 列表样式 (List)
- 列表样式：none（移除默认符号）
- 左内边距：10px
- 列表项：
  - 下边距：10px
  - 左内边距：20px（缩进）
- 特点：简洁缩进式列表，适合移动端阅读

### 页脚 (Footer)
- 居中对齐
- 上边距：50px
- 上边框：1px solid #eee
- 上内边距：20px
- 文字颜色：#999
- 字号：12px

## 5. Layout Principles

### Spacing System
- 基础单位：约 5px
- 常用值：4px, 5px, 8px, 10px, 12px, 15px, 20px, 25px, 30px, 40px, 50px

### Container
- 最大宽度：720px
- 外边距：0 auto（居中）
- 内边距：20px
- 背景：白色
- 阴影：`0 0 20px rgba(0,0,0,0.05)`

### Section Spacing
- 章节标题上边距：40px
- 章节标题下边距：20px
- 段落/组件间距：20-30px
- 头部下边距：30px
- 页脚上边距：50px

### Whitespace Philosophy
- **卡片浮动效果**: 720px容器 + 浅灰背景 + 阴影，内容如卡片浮于页面
- **章节呼吸**: 40px的章节间距确保内容分区明确
- **行间宽松**: 1.75行高适合中文长文阅读

### Border Radius Scale
- 微圆角 (2px): 章节标题装饰色块
- 小圆角 (4px): 概念框、公式框、标签
- 中圆角 (8px): 头部横幅、图片容器、表格、风险提示框

## 6. Depth & Elevation

| Level | Treatment | Use |
|-------|-----------|-----|
| Level 0 | 无阴影 | 页面背景 |
| Level 1 | 柔和阴影 `0 0 20px rgba(0,0,0,0.05)` | 内容卡片容器 |
| Level 2 | 微阴影 `0 2px 8px rgba(0,0,0,0.05)` | 表格 |

**Shadow Philosophy**: 阴影使用克制，主要功能是区分层级而非营造立体感。卡片容器的淡阴影足够让内容区域从背景浮起，表格的微阴影则增加精致感。

## 7. Do's and Don'ts

### Do
- 使用绿色(#05c15f)作为主色调，金色(#fdbb2d)作为强调色
- 为章节标题添加竖向色块装饰
- 使用左侧色块边框的强调框突出核心概念
- 表格采用渐变表头 + 斑马纹行
- 关键词使用圆角标签高亮
- 重要提示使用专用颜色区分（金色概念框、绿色公式框、红色风险框）
- 图片设置固定高度配合 object-fit: cover
- 保持1.75行高确保中文阅读舒适

### Don't
- 不要在主内容区使用纯黑文字(#000)，应使用#333获得更柔和效果
- 不要省略章节标题的装饰色块，它是重要的视觉锚点
- 不要使用过于鲜艳的配色组合，保持金融内容的专业感
- 不要让表格缺失阴影和圆角，这些细节提升精致度
- 不要忽略风险提示框的视觉设计，它是内容的重要组成部分
- 不要使用默认列表样式，缩进式列表更适合移动端

## 8. Responsive Considerations

### 移动端适配
- 内容容器720px最大宽度自动适应窄屏
- 内边距20px保证边距充足
- 图片高度200px适合竖屏浏览
- 表格宽度100%自适应

### 内联CSS注意事项
- 微信公众号过滤 `<style>` 标签，所有样式必须内联
- 使用语义化HTML结构
- 避免使用JavaScript交互
